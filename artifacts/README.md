# Artifacts

This folder contains generated reports produced by the analysis scripts.

Generated files:
- `summary.csv`: machine-readable run summary
- `summary.md`: markdown table summary
- `top_runs.md`: top N runs by a simple objective (demo ranking)

Plots:
- `temp_profiles.png`: Final timestep temperature profiles for the top runs
- `metric_sweep.png`: Scatter plot of energy_like_metric vs alpha (grouped by nx)
